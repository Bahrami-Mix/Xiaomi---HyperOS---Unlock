راهنمای اجرای اسکریپت در ترموکس

۱. بروز رسانی ترموکس:
pkg update && pkg upgrade -y
۲. اجازه دسترسی به حافظه داخلی (بعد از اجرا Allow بزنید و ترموکس را ببندید و دوباره باز کنید):
termux-setup-storage
۳. نصب پایتون:
pkg install python -y
۴. نصب کتابخانه‌های لازم:
pip install requests ntplib pytz urllib3 icmplib colorama
۵. رفتن به پوشه اسکریپت
(قبل از این کار فایل ZIP را از حالت فشرده خارج کنید و پوشه Unlock را به پوشه Download انتقال دهید):
cd /sdcard/Download/Unlock
۶. اجرای اسکریپت:
python Unlock\ premission.py
⚠️ توجه مهم قبل از اجرای دستورات:
  • اول فایل ZIP را از حالت فشرده خارج کنید.
  • پوشهٔ داخلی (مثل Unlock یا SCRIPT) را به پوشهٔ Download منتقل کنید تا مسیر دقیقاً مطابق دستور بالا باشد.
  • اگر پوشه را جای دیگری گذاشتید، کافی است در دستور cd مسیر صحیح را خودتان وارد کنید (مثلاً: cd /sdcard/YourPath/Unlock).
